The future of San Andreas DOJRP

Get the server to beta *W.I.P*

Many many spreadsheets, thousands of lines of code and a lot of testing later. we will have a fully featured server. The beta phase will allow us to see if the server works properly over time.

Leave Beta, Launch!

All stats are wiped and reset. Those players who played during the beta will receive a starting package as a thank you. At this stage we will probably throw a party for getting this far.

Add new features, review & tweaks

With new ideas and new people coming in, we can make the server even better with fresh content and by this time, the FiveM team will have made even more progress.

Build the next generation

Who will be the next PD chief? The next server owner? Ultimately we want this server to last a long time and that means building up new people to replace the existing team.
